# Go-Web-Template

**Go-Web-Template** 是一款基于Golang API微服务脚手架工程（模板工程）

版本最低支持，需求Go 1.13+

## 目前支持特性

基于此脚手架可以快速构建基于HTTP RESTFul的业务微服务

- 基于uber-zap 与 lumberjack.v2 日志模块
- 基于gin路由模块封装
- 基于gin数据绑定封装
- 基于validator支持中文校验封装
- 抽取API公用response简化代码
- 封装yaml.v2读取配置文件组件
